# 🤖 Customer Service Chatbot

A simple customer service chatbot built using **Flask** and **Machine Learning (Naive Bayes)**.

---

## 🚀 Features

* 💬 Web-based chatbot interface
* 🧠 Intent detection using ML (Naive Bayes)
* 🔁 Rule-based fallback system
* ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

* Python
* Flask
* Scikit-learn
* HTML, JavaScript

---

## 📂 Project Structure

```
customer-service-chatbot/
│
├── app.py
├── requirements.txt
└── templates/
    └── chat.html
```

---

## ▶️ Run Locally

### 1. Clone the repository

```
git clone https://github.com/dileepreddy2003/customer-service-chatbot-base.git
cd customer-service-chatbot-base
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the app

```
python app.py
```

### 4. Open in browser

```
http://127.0.0.1:5000
```
